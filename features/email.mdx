## Keyboard Shortcuts

Macro's email client is designed to be operated entirely from the keyboard:

| Action | Shortcut | Description |
|--------|----------|-------------|
| Archive | `e` | Archive the current thread (or unarchive if already archived) |
| Reply | `Opt+R` | Reply to the current thread |
| Reply All | `r` | Reply to all participants in the thread |
| Forward | `f` | Forward the current thread |
| Next Message | `↓` | Navigate to the next message in the thread |
| Previous Message | `↑` | Navigate to the previous message in the thread |

<Tip>
Every action in Macro has a keyboard shortcut. You never need to reach for your mouse to triage email.
</Tip>

## Thread View

Emails are organized by thread, showing the full conversation history. Each thread displays:

- All messages in chronological order
- Sender information and timestamps
- Attachments (automatically imported to file storage)
- Quick actions for archiving, replying, and forwarding

<Note>
Attachments from emails are automatically imported into Macro's file storage system, making them searchable and linkable from anywhere in your workspace.
</Note>

## Composing Email

When composing email, you have access to Macro's full rich text editor:

- **Rich formatting**: Bold, italic, lists, and more
- **@mentions**: Reference documents, tasks, or other content
- **File attachments**: Drag and drop files or select from your storage
- **Inline images**: Embed images directly in your message

## Integration with Macro

Email threads can be:

- **@mentioned** in documents, channels, and canvas
- **Searched** alongside all other content
- **Linked** bidirectionally to create traceable references
- **Shared** with team members through channels

<Info>
When you @mention an email thread in another part of Macro, it creates a bidirectional link you can trace back from the email itself.
</Info>

## Inbox Management

Stay on top of your email with:

- **Inbox filtering**: View only unarchived threads
- **Search queries**: Find emails with `type:email inbox:true`
- **Unified view**: See emails alongside tasks, messages, and documents
- **Smart archiving**: Remove clutter while keeping everything searchable
